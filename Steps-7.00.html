How to use different feature of Query Editor/Transform . 7

Now since our data is loaded, we will make some modifications on the Query editor/Transform.

1/Open data view

2/Click Transform data/ Query editor 


Filter

Filter out all country except USA since our report is about USA only.

1/Click on the filter icon

2/High light USA only


Replace value

We must replace the null values in all columns by 0.
1/Right click on the column

2/Click replace value

3/Type null on the value to find box and type 0 on the replace value box.

4/Do the same process for the other columns

 



Duplicate column 

We will duplicate primary fuel column then we are going to replace (biomass. coal , co-generation, gas, geothermal ,nuclear, oil,petcock,storage,waste and others ) by thermal plants and ( wind, solar , hydro) by renewable. 

1/Right click on the primary fuel column then click duplicate

2/Go to the duplicate column make a right click

3/Click replace value

4/Type one by one (biomass. coal, co-generation, gas, geothermal ,nuclear, oil, petcock ,storage ,waste and others ) on the value to find box and type thermal plants on the replace value box.

5/Type one by one (  wind, solar , hydro) on the value to find box and type renewable on the replace value box.

6/Finally click close &apply
